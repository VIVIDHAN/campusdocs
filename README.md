# CampusDocs 📚

CampusDocs is a web platform designed to help students access campus-related academic resources easily.

## 🚀 Features
- Organized academic resources
- Easy navigation
- Responsive UI
- Built with modern frontend tools

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Vite
- Node.js
